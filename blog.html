Question-1: What are the different ways to select an element in the DOM?

There are several ways to select an element in the DOM using JavaScript. Here are some of the most frequently used methods:

  • getElementById()
  • getElementsByClassName()
  • getElementsByTagName()
  • querySelector()
  • querySelectorAll()
  • closest()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML retrieves or sets HTML content, including tags, while innerText gets only visible text, ignoring hidden elements. textContent extracts all text, including hidden content and this is the fastest option.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where a parent element handles events for its child elements using event bubbling. Instead of adding event listeners to multiple child elements, a single listener on the parent can efficiently manage interactions.

Question-4: What is event bubbling in the DOM?

Event bubbling is a mechanism where an event starts from the target element and propagates up through its ancestors in the DOM. This allows parent elements to capture and handle events triggered by their child elements.

Question-5: How do you create, add, and remove elements using JavaScript?

I can create a JavaScript element with document.createElement(), add it using appendChild() or append(), and remove it with removeChild() or remove().
